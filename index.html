Project Idea

USB 3 Transmiter/Reciever

I believe that there is a need in the computer accessories market for a peripheral that transmits data at USB 3.0 speeds (5Gbps). This device at a basic level will consist of two components. A "Transmitter" and a "Receiver". The "Transmitter" will have a USB female port to plug your USB device into, it will also have a power pack that will power the device for and the attached USB device for at least 4 hours, the transmitter in the transmitter device will be the new WiFi 6. The "Receiver" will plug into a PCI slot on your computer and will operate a wireless network, the software included will allow for customization of the network and allow multiple devices to be added.

My personal motivation behind this idea is the fact I am a avid gamer and have a lot of peripherals and the cable mess is nightmarish. I also enjoy VR, the Oculus VR system uses camera sensors that perform best at USB 3.0 speeds, but in order to get room scale game play you need to run 5-10 meters of USB extension cables, which provide a tripping hazard and technical difficulties trying to get USB 3 speeds, as you will need to provide the cables with external power sources. I believe that this Project Idea will be very useful in this situation.

As stated above I believe this Project Idea will consist of two (2) main hardware components and at least 1 major software component. The wireless technology (WiFi 6) for this Project Idea here fore referred to as "Project" is not available as it is still in development. Wifi 6 is predicted to have WiFi speeds of 9.6Gbps which will allow up to two (2) USB 3.0 devices to be connected as USB 3.0 has speeds of 5Gbps. The transmitter Device will require a power pack built into it I believe that a detachable and separate recharging dock would be a good business choice as it will allow the battery pack to be upgraded with new battery technologies as time progresses. The main technical issue in regards to the batteries is working out how much power the system will require to operate for four (4) hours while providing power to a connected device and powering the WiFi 6 hardware.

The Receiver which will be inserted into a PCI Express 4.0 slot as this will allow for massive data transfer speeds allowing the system to handle multiple Wifi 6 incoming signals from Transmitters. Due to the fact that WiFi 6 appears to handle only two (2) USB 3.0 amounts of data (9.6 Gbps) we would need to place at least two (2) WiFi transmitters on the PCIE Card which will allow 19.2 Gbps transfer speeds which will allow us to handle four (4) USB 3.0 connections which will make the Project more commercially attractive. The software will be required to be able to load balance between the two (2) transmitters in order to provide the best possible connection with minimal latency. The software will need a user interface that will allow users to add the wireless USB Transmitters with minimal fuss. Security will be an issue for the users so the software will need to allow us to secure the Wifi Signals.

In order to complete the Project we will require skills in the following areas; Hardware Engineering: A Hardware Engineering will be required to create a small package device with an external power pack while also providing sufficient room to allow circuitry for a USB 3.0 type A female connection point and a Wifi 6 transmitter and any cooling that may be required. The hardware engineer will also need to create the PCIE card that has 2 Wifi 6 transmitters.

Software Engineering: A Software Engineer will be required to create the software for both the transmitter and the receiver while also creating the user interface software.

If we can manage this Project we will solve every gamers issue of peripheral cable nightmare as it is solved by the use of wireless technology, if we make our transmitter software open sourced it will allow other peripheral companies such as Logitech to make wireless peripherals that will use our product which will greatly increase the appeal behind the Project.

This will solve a lot of the peripheral issues that most true gamers have.
